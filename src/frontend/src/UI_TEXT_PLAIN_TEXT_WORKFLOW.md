# UI Text Plain-Text Workflow

This document explains how to edit the UI text translations for the A&A Boxes application.

## Overview

All user-facing UI text is stored in a single plain-text file using a simple key-value format with bilingual support (English | Spanish).

## File Location

**Repository source file:** `frontend/src/texts/ui-texts.txt`

This file is the single source of truth for all UI translations in the application.

## Downloading the Translations File

You can download the current translations file directly from the application UI:

1. Scroll to the **footer** at the bottom of any page
2. In the **Quick Links** section, click the **"Download UI Texts"** button (with download icon)
3. Your browser will download a file named **`ui-texts.txt`**

The downloaded file contains the exact same content as the repository source file, including all comments and section headers.
